System Manager - Network
Network Section [Top]

Network provides the functionality to modify the network settings.

The Command Format:

  • CMD_NETWORK_CONF     temporal_network_xml_file     END_OF_COMMAND

    • SysMgr parses the temporal_network_xml_file and write back it to Kilrogg' network xml configuration file (/mnt/flash/etc/network/network_conf.xml in default). Then rewrite the linux system configurations files according to the new configurations. The configuration files includes /etc/network/interfaces, /mnt/flash/etc/ppp/pap-secrets, /mnt/flash/etc/ppp/pppoe.conf and /mnt/flash/etc/boa.conf.
    /etc/network/interfaces Settings related to the network interfaces.
    /mnt/flash/etc/ppp/pap-secrets The login username and password for PPPoE.
    /mnt/flash/etc/ppp/pppoe.conf PPPoE configuration file.
    /mnt/flash/etc/boa.conf Boa http server configuration file.
