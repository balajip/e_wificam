System Manager - Syslog
Syslog Section [Top]

Syslog provides the functionality to modify the remote syslog settings.

The Command Format:

  • CMD_SYSLOG_CONF     temporal_syslog_xml_file     END_OF_COMMAND

    • SysMgr parses the temporal_syslog_cml_file and write back it to Kilrogg' remote syslog xml configuration file (/mnt/flash/etc/conf.d/remote_log_conf.xml in default). Then rewrite the linux system configurations files according to the new configurations. The configuration files includes /mnt/flash/etc/services and /mnt/flash/etc/syslog.conf.
    /mnt/flash/etc/services Set the remote syslog server's port.
    /mnt/flash/etc/syslog.conf Determine the warning level, remote server address and the output log file.
