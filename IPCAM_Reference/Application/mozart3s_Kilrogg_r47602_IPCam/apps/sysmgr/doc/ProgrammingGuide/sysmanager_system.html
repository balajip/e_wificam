System Manager - System
System Section [Top]

System provides the functionality to modify the system time settings.

The Command Format:

  • CMD_SET_SYSTEM     temporal_system_xml_file     END_OF_COMMAND

    • SysMgr parses the temporal_system_xml_file and write back it to Kilrogg' system xml configuration file (/mnt/flash/etc/conf.d/system.xml in default). Then we call the system calls to modify the system settings. The system calls includes : /usr/sbin/setntpcron, /etc/init.d/bbcrond and /etc/init.d/heartbeat.
    /usr/sbin/setntpcron Reads the ntp settings from /mnt/flash/etc/conf.d/system.xml and modify the crontab.
    /etc/init.d/bbcrond Restart the crond.
    /etc/init.d/heartbeat Turn on/off the led according to the led setting in /mnt/flash/etc/conf.d/system.xml.
