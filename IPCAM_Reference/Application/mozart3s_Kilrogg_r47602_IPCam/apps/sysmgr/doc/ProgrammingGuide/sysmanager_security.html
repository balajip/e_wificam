System Manager - Security
Security Section [Top]

Security provides the functionality to modify user's password.

The Command Format:

  • CMD_SYS_SECURITY     username     password     END_OF_COMMAND

    • SysMgr reads user's password line by line from the password file (/etc/passwd in default). Then write the user's password to a new file (/tmp/passwd.new). If the user name matches the user name that we want to modify, we compute its encrypted password by the crypt() function. Finally, we move the new password file to replace the old one.
