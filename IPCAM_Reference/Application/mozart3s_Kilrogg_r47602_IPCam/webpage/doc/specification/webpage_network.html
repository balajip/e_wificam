Functions [Top]

This page includes network related settings.

Network type

Network type set the connection type LAN or PPPoE. If there is a DHCP server, automatically get IP address is to request and register an IP from DHCP server.

If a fixed address is assigen by ISP or LANs manager, user also can set the fixed IP address manually.

If a ISP provides name and password PPPoE connection, fill the PPPoE interface section to connect to internet.

HTTP

This field is to set HTTP server. The only changable server properties is connection port. Since we provids the motion JPEG throgh the HTTP server. The motion JPEG access name should be modifiable, the user can access the motion JPEG by connect to "http://IP_address:Port/Accessname"

FTP

The system also provides a FTP server to update firmware. User should able to change the FTP connection port.

RTSP Streaming

The MPEG-4 video is streaming by RTSP server, the access name should be modifiable. The user can access the streaming by connected to rtsp://IP_address:Port/Accessname by other player. Since the RTSP technique is connect RTCP and RTP for audio and video respectively, these connection port should be changable for security issue.
