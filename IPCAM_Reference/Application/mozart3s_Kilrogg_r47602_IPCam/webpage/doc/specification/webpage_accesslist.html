Functions [Top]

Access list is to control the access permission of clinets by check the IP address.

There are two IP lists, allow list and denied list. Only those client whose IP is in allow list and not in denied list can access server. Each list can have 10 entries. Type start and end adress in text and click add button to add a new list. To delect a exist list, select the list and click delete button.
