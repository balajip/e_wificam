The plugin provides to select motion window on video. To create a new window, click new button and select detection range and assign window name, then click save button to save. Total tree windows are aviable. User can disable motion detection by uncheck the enable motion detection box. The windows information should be saved even the motion detection is disable. To delete a window, just click "x" button on the window.

Tow properties are assign to each window, sensitiveity and percentage. The sensitiveity means how larg object action will be detected as motion and the percentage meas that the area is detected to have motion in a windows is over pecentage will triggle the motion alarm.
