Functions [Top]

Since there are multi streamings, for IE or other browsers, the stream options set which stream to connect in the Display page.

If the connection by IE browser, the MPEG-4 connection setting is display. Media options set connect video only or audio only or both.

The protocol set the option set the connection protocol for MPEG-4 streaming.

  • UDP: RTSP over TCP for signaling, RTP/RTCP for media streaming.
  • TCP: RTP/RTCP over RTSP for media streaming.
  • HTTP: RTP/RTCP over RTSP, RTSP over HTTP for signaling and streaming.

Note: If the streaming setting is MJPEG, both media options and protocol options are ignore.
Note: For other browsers, the QuickTime plug-in is used, since QuickTime didn't provide the setting for connection protocol or media options, there are streaming options only.
Note: All these settings should be register as cookies in the client computer.
